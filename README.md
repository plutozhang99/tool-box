# Tool Box

A collection of personal utility scripts and AI coding-agent workflows for everyday automation tasks.

Includes standalone Python scripts for quick jobs plus agent workflow docs and skills:

- `claude-code-tools/` - original Claude Code skills and notes
- `codex-tools/` - Codex-translated skills and notes

The Codex tools do not depend on `everything-claude-code`. Their phase review flow uses a local Codex review bundle: code review, security review, functional coverage, and project-specific verification commands.
